Support and management tools

Ubuntu Advantage is the commercial support package from Canonical. It includes Landscape, the Ubuntu systems management tool, and the Canonical Livepatch Service, which enables you to apply kernel fixes without restarting your Ubuntu {{lts_release_full}} systems.

    Always on access to the world’s leading Ubuntu desktop, server and cloud experts, all actively participating in open source projects.
